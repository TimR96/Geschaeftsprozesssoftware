Geschäftsprozesssoftware, bestehend aus einem Java-Programm, einer Website und einer Datenbank für einen Friseursalon,
mit der verkaufte Dienstleistungen und Produkte in einer Datenbank gespeichert werden können.
Diesen wird ein Betrag, eine Zahlungsart, eine Kategorie und ein Mitarbeiter zugeweisen. Die Kategorien und Zahlungsarten
können vom Anwender geändert werden. 
Der Bargeldbestand der Kasse wird getrackt. Er kann manuell angepasst werden.
Mitarbeiter können hinzugefügt, gelöscht und mit einem RFID-Chip verknüpft werden.
Die Tagesumsätze und der Tagesgesamtumsatz kann eingesehen werden.
Mitarbeiter können ihre Kommen- und Gehenzeiten im System buchen.
Die RFID-Erkennung kann aktiviert und deaktiviert werden.
Es kann ein Monatsumsatzbericht als PDF erzeugt werden.
